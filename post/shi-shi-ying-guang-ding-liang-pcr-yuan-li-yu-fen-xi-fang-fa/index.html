实时荧光定量PCR原理与分析方法

2021-03-29 / 5 min read

荧光定量PCR概述

如何实现监控信号

  • 在PCR体系中添加某种可发光的物质
  • 信号检测仪器 -qPCR仪
    **化学原理:**荧光报告分子是检测的基础
SYBR Green染料法 Taqman探针法
结构 嵌合型核酸染料 5'R---Q3‘
优势 成本低、有溶解曲线生成 特异性高、低丰度基因定量可信度高
劣势 特异性较Taqman低 成本高 引物二聚体的形成，无法直观判定
应用 SNP分型、表达量分析、基因突变检测、病原菌检测 病毒检测、病毒载量检测、基因分型、 基因突变检测、SNP分型、表达量分析等

光信号、数字信息处理的可靠性

  • 特异性分析

  • 溶解曲线

  • 避免非特异性扩增，单一产物

  • 标准曲线分析
  • 扩增效率、线性检测范围、检测下限、R2、重复性
  • 仪器设置
  • 基线、阈值
  • Rox Reference Dye

qPCR能够解决什么问题

  • 定性研究
  • 通过溶解曲线或探针判定扩增子产物类型，可用于基因分型或SNP分型等。
  • 定量研究
  • 相对定量:基因表达量差异，如测序数据的验证
  • 绝对定量:基因拷贝数/基因浓度，病毒滴度测定

qPCR有别于PCR之处

  • qPCR技术的出现是为了解决常规PCR无法解决不了的问题_基因定量
  • qPCR有别于PCR基因定量的真实依据是:指数扩增期。
  • 术语:扩增曲线、溶解曲线、标准曲线、Ct值、Tm值等。

引物设计

  • 引物扩增效率评估
  1. 标准品制备
  2. 标准品稀释
  3. qPCR扩增
  4. 标准曲线

  • NTC评估引物二聚体

    Tips: NTC污染解决方案

    1. 确认NTC信号来源: 引物二聚体、体系中污染靶标分子
    2. 解决方案 调整引物浓度、单一变量确认污染源
    3. PCR标准: NTC熔解曲线和扩增曲线无荧光值变化，即Ct值为0，无扩增出现。 若有污染，可接受范围: Ct(NTC) - Ct(最低浓度)25。根据经验，NTC Ct值常见范围: >30

引物设计原则

  • 扩增效率相关要求
  • 扩增子大小80-200bp
  • 引物长度20bp左右
  • GC含量40-60%
  • 避免出现连续4个相同碱基
  • 避免引物出现二级复杂结构
  • 3'端G
  • 上下游引物Tm值接近，建议60’C+-2C
  • 特异性相关要求
  • 设计跨内含子引物
  • 避免引物3'端出现互补
  • Blast保证保守性
  • 相关软件推荐:
  • Primer3
  • MFold (http://mfold.rna.albany.edu)
  • BLAST (http://blast.ncbi.nlm.nih.gov/)

RNA质量

  • RNA完整度
  • RIN值 (RNA integrity number ): RIN>7为高质量，6-7勉强可用，<6为较差质量。
  • 该值可用Agilent 2100 Bioanalyzer仪器去检测

  • RNA纯度: gDNA污染问题

    Tips: NRC染解决方案

    1. 确认NRC信号来源: RNA提取过程中污染gDNA
    2. 解决方案：RNA电泳，判定RNA质量;逆转录之前或过程中，或定量过程中，去除gDNA污染。
    3. qPCR标准;不能容忍有gDNA污染

逆转录

  • RNA二级结构、逆转录酶、逆转录引物

    1. 逆转录酶对cDNA产量的影响
    2. 具备耐热性能的逆转录酶可用于复杂二级结构RNA和一步法 RT-qPCR
    3. 逆转录引物对cDNA产量的影响

  • 逆转录效率

  • 逆转录基因的均一性
  • 线性检测范围
  • R2
  • 灵敏度

Ct值的计算

  • 相对定量原则:内参基因和目的基因的扩增效率接近，且接近100%。
  • 第1种情况. 内参和目的基因扩增效率一致，且接近100%
  • 第2种情况. 内参和目的基因扩增效率有差异

操作流程

RNA样品——gDNA去除后的RNA——cDNA——q PCR

常见问题

问题1:筛选qPCR试剂，唯C值论?

  • 试剂性能评测指标

样本类型、基因长度和GC合量、数量

  • 检出率
  • 特异性
  • 扩增效率与灵敏度:高、中、低表达量、内参
  • 重复性
  • 精准性
  • 体验度指标
  • 操作简便性:组分情况、颜色、适用仪器、体系配制、引物设计
  • 反应时间
  • 试剂稳定性
  • 重复性
  • 体系稳定性

问题2: Ct值偏大

  • 解决方案:

    1. 提高模板浓度，或重新制备模板 (模板降解)
    2. 优化扩增程序，或重新设计引物(引物扩增效率低)
    3. PCR产物在100-300bp之间(扩增效率低)
    4. 提高模板稀释倍数或重新制备模板(模板质量低)
